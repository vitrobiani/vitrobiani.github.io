Project For System Progeaming Class

Overview

This was a project made in C as a part of a system programing class.

The project depicts a department store chain management system.
This project is not connected into any database so the information is saved on a binary file in the projects directory.

Project's Github

Features

Probably all the features you would think to find in a program like this.

  • adding a store to the chain
  • adding products to sell
  • focus on any spesific store and get its info
  • adding employees to a store
  • updating product stock
  • getting the best selling product
  • getting the best employee

    • Loading the project

    This project was written in Visual Studio environment, so to load it just clone the repo there for best results.
